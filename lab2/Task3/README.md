# Задание 3. Число инверсий.
## Описание
В этом задании студенты подсчитают число инверсий массиве.
## Структура проекта
- `src/` — исходный код программы и ввод, вывод данных.
- `tests/` — автоматические тесты для проверки работы кода.
## Запуск проекта
1. Создать файл кода и файл ввода, вывода данных.
2. Написать программу.
3. Использовать функцию `bubbleSort`
## Тестирование
1. Написать код в файле теста.
2. выплнить программу.
